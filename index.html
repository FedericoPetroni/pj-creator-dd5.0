Generador de Personajes de Dungeon & Dragons versión 5.0

Bienvenidos a mi proyecto de Generador de Personajes para D&D

El generador será un proyecto a largo plazo. En primera instancia se generará de tal forma que aleatoriamente genere un documento que incorpore los atributos más básicos de los personajes más simples del proyecto. Esto se hace en pos de aprender y afianzar conocimientos de lenguajes de programación.

Se irá documentando el proyecto en sí. Por tanto es probable que mute constantemente para dar una hoja de trabajo del mismo. El repositorio completo se subirá a GitHub para su muestra al público (probablemente).
